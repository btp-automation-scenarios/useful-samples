# Useful Terraform Samples for SAP BTP

## Overview

This repository contains a collection of Terraform code samples designed to help you deploy and manage SAP Business Technology Platform (BTP) resources efficiently.

You might run into some specifics of SAP BTP when trying to set up your Terraform configuration. This repository provides sample solutions for use cases brought to our attention by users of Terraform on SAP BTP.

## Repository Structure

Each folder contains a solution proposal for a specific use case. Each folder includes:

- A `README.md` file that describes the problem statement, the proposed solution, and any known restrictions.
- Terraform code files that implement the proposed solution.

> [!CAUTION]
> Use the samples at your own risk. They are provided "as-is" without any warranties or guarantees. Always review and test the code in a non-production environment before deploying it to production.
